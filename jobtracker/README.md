This is a very basic chrome extension that is simply going to take any time you click apply in LinkedIn or Indeed and
move it to a google sheets dump. 

HOW TO USE:
1. Download files from the repo.
2. Make a new google sheets, go to extensions -> Apps Scripts -> drop in sheets_script 
3. replace the replace section with your sheets ID found in your sheets URL
4. Deploy and copy the link
5. Replace webhook with the copied link then load extension in google extension
6. Should then automatically track anytime you apply for a new job

Data Disclaimer:
This extension stores job application metadata locally or sends it to a user-configured endpoint.
No data is sold or shared with third parties by default. No data from this is saved by me or any third party.
